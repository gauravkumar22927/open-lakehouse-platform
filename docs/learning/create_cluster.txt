You should see

Why Node Labels?

This is one of the first Kubernetes concepts interviewers appreciate.

Later we'll tell Kubernetes things like:

Spark Driver
      ↓

node-role=compute

or

MinIO

↓

node-role=storage

using nodeSelector or node affinity. That lets us control where workloads run.
